Como estilizar a tag mark com CSS?

É possível estilizar as tags mark individualmente, ou de toda a página HTML. Para alterar de toda a página, é só adicionar uma regra de estilo no <head> da página.

Exemplo:
Texto marcado individualmente 1
Texto marcado individualmente 2
Texto marcado individualmente 3
Texto marcado pelo head
Texto marcado pelo head
Texto marcado pelo head
